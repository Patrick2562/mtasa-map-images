# MTA:SA Map Images
The maps were generated in-game with an orthographic shader to eliminate perspective distortion and achieve a true top-down representation.


## Resolutions
- 1024px, 1536px, 2048px, 4096px, 8192px, 10240px, 20480px

**Recommended for making an in-game minimap: 2048px or less**
